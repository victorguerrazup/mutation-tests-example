# Mutation Tets Example

## Requirements
- NodeJS 16.14.0+
- NPM 8.3.1+

## Install Dependencies
```shell
npm install
```

## Run tests
```shell
npm test
```

## Run Code Coverage
```shell
npm run test:coverage
```

## Run Mutation Tests
```shell
npm run test:mutation
```

Mutation HTML report will be located at 'reports/mutation/html/index.html'
